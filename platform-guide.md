# Platform Guide: X (Twitter) & TikTok

## X (Twitter) — Rules & Limits

| Limit | Value |
|-------|-------|
| **Tweet length** | 280 characters max |
| **Thread** | Use multiple tweets (1/X format) to go longer |
| **Images** | Up to 4 images per tweet |
| **Video** | Up to 140 seconds (2:20) |
| **Schedule** | 2-3 tweets/day, space 30min apart |

### Thread Best Practices
- Start with hook (🧵 indicator)
- Number posts: 1/X, 2/X, 3/X...
- Keep each post under 280 chars
- End with CTA or "Follow for more"

### Posting via Post-Bridge
```
Create each tweet as separate draft:
- Post 1: Hook + "🧵" thread indicator
- Post 2-N: Continue thread
- Last post: CTA (DM, follow, link)
```

## TikTok — Rules & Limits

| Limit | Value |
|-------|-------|
| **Video length** | 10 sec - 10 min (3 min default) |
| **Caption** | Up to 2,200 characters |
| **Hashtags** | Up to 100 (but 3-5 is best) |
| **Schedule** | 1-2 videos/day, space 2hrs apart |

### Content Structure
- **Hook (0-3 sec):** Grab attention immediately
- **Body (3 sec - end):** Deliver value
- **CTA:** "Follow for more" / "Save for later"

### Posting via Post-Bridge
- Draft posts without media first
- **Manually attach media in Post-Bridge UI** — API upload not supported
- Use thumbnail from `assets/images/` folder

---

## Common Mistakes to Avoid

| Mistake | Fix |
|---------|-----|
| Posting single tweet over 280 chars | Split into thread |
| No hook in first 3 seconds | Lead with pain/curiosity |
| Too many hashtags (TikTok) | 3-5 max |
| Posting all content at once | Space 30min (X) / 2hrs (TikTok) |
| Forgetting CTA | Always end with follow/save |
